Detailed Description

ECDSA-in-SNARK prover witness (eprint 2025/695 §5).

Contains every intermediate value required by a PLONK circuit to verify one secp256k1 ECDSA signature using foreign-field arithmetic. Values are provided in canonical 32-byte big-endian encoding AND in 5×52-bit limb form (ufsecp_ff_limbs_t) for direct PLONK gate wiring.

ECDSA verify circuit steps: s_inv = sig_s^{-1} mod n u1 = msg * s_inv mod n u2 = sig_r * s_inv mod n R = u1*G + u2*pubkey valid = (R ≠ ∞) AND (R.x mod n == sig_r)
