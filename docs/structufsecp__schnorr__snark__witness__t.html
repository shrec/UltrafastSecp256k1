Detailed Description

Complete PLONK prover witness for one BIP-340 Schnorr verification.

BIP-340 verify steps: R = lift_x(sig_r) (even Y) P = lift_x(pubkey_x) (even Y) e = H("BIP0340/challenge" || R.x || P.x || msg) mod n valid = (s*G == R + e*P) AND (R.y is even)
